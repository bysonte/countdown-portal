# Punta Cana 2026 - Countdown Portal

Un portal interactivo premium diseñado para acompañar la cuenta regresiva del viaje a Punta Cana (Sep 2026).

## 🚀 Características Actuales

- **Hero & Countdown**: Temporizador dinámico segmentado por bloques con diseño responsivo escalable.
- **Collage Dinámico**: Fondo animado con transiciones suaves en cuadrícula de fotos optimizada.
- **Galería interactiva**: Grid de imágenes cargadas con soporte para zoom en pantalla completa (Lightbox).
- **Control Musical**: Reproductor de música integrado y visualizador de espectro interactivo.
- **Ubicación en vivo**: Mapa híbrido apuntando directamente al hotel sede.
